ICCAT cloud infrastructure

Summary

The ICCAT Cloud Infrastructure is currently under development (all 2013).Its is made of 3 cloud servers deployed in Rackspace data center (UK). All servers were deployed using Ubuntu Linux server flavours. And all the services deployed will heavily be based in FOSS (free open source software). This server in particular should be basically used for scientific calculations.

This server (currently under development) is used to run RStudio server

Login to RStudio Server

For additional information send an email to Laurie Kell (laurie.kell@iccat.int)


Services deployed so far:

  • RStudio Server
  • Apache web server (a special web page for each registered user)
  • Secure FTP access (to each users space)
  • onwnCloud (under configuration/testing)
